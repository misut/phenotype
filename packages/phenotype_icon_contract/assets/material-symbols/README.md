Material Symbols assets
=======================

These SVG files are downloaded from Google's Material Design Icons repository
and pinned to revision `4d7678801370dc2fe9c35b437570f56f56e43801`.
Phenotype embeds Material Symbols (new) in the `outlined`, `rounded`, and
`sharp` styles; `outlined` is the default runtime style.

Source: https://github.com/google/material-design-icons
License: Apache License 2.0

The checked-in SVGs are normalized for phenotype rendering by adding
`fill="currentColor"` to the root `<svg>` element. No icon path data is
hand-authored in this repository.
